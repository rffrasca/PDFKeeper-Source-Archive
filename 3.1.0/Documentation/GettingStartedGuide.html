Introduction

This document describes how to get started with PDFKeeper.

PDFKeeper can be implemented for use by a single user or for use in a multi user environment. If implementing for use in a multi user environment, it is recommended that the Oracle XE Database be installed on a dedicated system.

Getting Started

How can I get started using PDFKeeper?
  1. Please read the release notes for any late-breaking information.
  2. If new to Oracle XE, read the Oracle Database Express Edition documentation for instructions on how to install, configure, and administer an Oracle Express Edition Database.
  3. For the system that will host the database, make sure the following software is installed:
    • Oracle Database Express Edition
  4. For each system that the PDFKeeper application will be used on, make sure the following software is installed:
    • ODP.NET, Managed Driver. For installation instructions, refer to the Readme that is included with the Xcopy ZIP file. Note, when installing on a UAC (User Access Control) enabled Windows system, make sure to perform the installation with a Command Prompt session that was opened using Run as administrator.
  5. Refer to the Database Setup and User Account Management Guide for configuring the PDFKeeper database schema. Note, if PDFKeeper was just upgraded from version 1.0.0 or 1.1.0, please refer to section 3. Database User Account Management for additional grants that need to be provided to existing database user accounts. No database configuration changes are required when upgrading from version 2.0.0 or above.

Uninstalling PDFKeeper

How can I uninstall PDFKeeper?
  1. As an Administrator, open Add or Remove Programs or Programs and Features.
  2. Select PDFKeeper from the list.
  3. Click Uninstall to start the uninstall.
  4. Click Yes.
  5. Close out of Add or Remove Programs or Programs and Features.
