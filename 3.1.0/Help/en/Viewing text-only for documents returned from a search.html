Viewing text-only for documents returned from a search

To view only the text of each document returned from a search without opening it using the bundled third-party viewer, select the document list view record on the Document Search tab, and then select the Document Text-Only View tab. The text from the PDF document will be extracted and displayed in the text box. This feature will allow for quick viewing and scanning through the result set.

The following functions can be performed using Document Text-Only View:

  • View extracted text for the previous or next document in the result set using the Previous or Next buttons; or you can use the keyboard shortcuts Alt+P or Alt+N.
  • Open the selected document using the bundled third-party viewer by double clicking on the text box that contains extracted text.
  • When switching back to the Document Search tab, the document that was having its extracted text viewed will be the selected record in the document list view on the Document Search tab.

Notes

  • Only document text that can be extracted will be displayed, text from images will not be extracted.
  • Each PDF document that text was extracted from (including image only PDF documents) will remain cached on disk until PDFKeeper is closed out.
  • If your operating system supports the Encrypting File System (EFS), each PDF document that text was extracted from (including image only PDF documents) will be encrypted. The following operating systems do not support EFS: Windows XP Home, Windows Vista Home Basic and Premium, and Windows 7 Starter, Home Basic and Premium.
