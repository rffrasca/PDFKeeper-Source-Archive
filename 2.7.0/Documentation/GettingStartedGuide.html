Introduction

This document describes how to get started with PDFKeeper.

PDFKeeper can be implemented for use on a single user system or in a workgroup. If implementing for use in a workgroup, it is recommended that the Oracle XE Database be installed on a dedicated system.

Getting Started

How can I get started using PDFKeeper?
  1. Please read the release notes for any late-breaking information.
  2. For new users': read the Oracle Database 11g Express Edition documentation or Oracle Database 10g Express Edition documentation for instructions on how to install, configure, and administer an Oracle Express Edition Database.
  3. For new users': on the system that will host the database, make sure the following software is installed:
    • Oracle Database 11g Express Edition (11.2) Server or Oracle Database 10g Express Edition (10.2.0.1) Server (Universal)
  4. For each system that the PDFKeeper application will be used on, make sure the following software is installed:
    • Oracle Data Access Components (ODAC) 11.2.0.2.1 or Oracle Data Access Components (ODAC) 11.2.0.1.2. Download and install the Oracle Universal Installer version; and when prompted, select Oracle Data Access Components for Oracle Client. Note, only the Oracle Data Provider for .NET and its dependency needs to be installed.
  5. Refer to the users' guide for configuring and using PDFKeeper. Note, if PDFKeeper was just upgraded from version 1.0.0 or 1.1.0, please refer to section 3. Database User Account Management for additional grants that need to be provided to existing database user accounts. No database configuration changes are required if upgrading from version 2.0.0 or above.

Uninstalling PDFKeeper

How can I uninstall PDFKeeper?
  1. As an Administrator, open Add or Remove Programs or Programs and Features.
  2. Select PDFKeeper 2.7.0 from the list.
  3. Click Remove to start the uninstall.
    1. Click Yes.
    2. Click Ok.
  4. Close out of Add or Remove Programs or Programs and Features.
