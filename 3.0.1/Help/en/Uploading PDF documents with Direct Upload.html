Uploading PDF documents with Direct Upload

PDF documents can be uploaded with Direct Upload using the following methods:

  • From Web Browser/application, save PDF document to the desired sub-folder in the DirectUpload folder using the PDFKeeper Direct Upload shortcut located in the My Documents folder on Windows XP or the Links folder on Windows Vista and later.
  • Configure Scanners/Document generator applications to save each PDF document to %USERPROFILE%\Local Settings\Application Data\PDFKeeper\DirectUpload\<sub-folder> on Windows XP or %LOCALAPPDATA%\PDFKeeper\DirectUpload\<sub-folder> on Windows Vista and later.

Note

  • The PDFKeeper Direct Upload shortcut is created when PDFKeeper is started and is removed when PDFKeeper is closed out.
  • While Direct Upload is uploading PDF document's, a green icon with an arrow pointing up will be visible on the right side of the PDFKeeper status bar. In addition, the Direct Upload Configuration menu item will be grayed out while uploading.
  • PDFKeeper Direct Upload will only accept PDF documents, all other file types will be ignored and remain in the DirectUpload\<sub-folder> until manually removed.
  • PDF documents that contain an OWNER or USER password are not supported and will remain in the DirectUpload\<sub-folder> until manually removed.
  • It is NOT recommended to configure the Windows Recycle Bin to delete files immediately as Direct Upload will delete all preprocessed PDF files to the Recycle Bin.
