Capturing HTML Documents

HTML documents can be captured using the following procedure:

  1. From PDFKeeper, select the Document Capture tab.
  2. Select the Tools menu, and then select HTML Converter; or you can use the keyboard shortcut Ctrl+H.
  3. Browse to the desired location.
  4. Select one or more HTML files to be converted and captured.
  5. Select Open.

Note

  • Each HTML document selected will be converted to PDF; each PDF document will be created in the Capture folder and will be added to the Document Capture Queue listview. The source HTML documents will remain in their original location.
  • This feature requires wkhtmltopdf to be installed. For instructions, please refer to the Getting Started Guide.
