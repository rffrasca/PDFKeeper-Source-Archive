Database Connection

  1. Enter the database Username.
  2. Enter the database Password.
  3. Enter the database Data Source. The Data Source must be specified as: host:port

Note

  • host is the host name or IP address of the computer that is running Oracle Database XE Server. If connecting locally to Oracle Database XE Server, the hostname or IP address would be localhost or 127.0.0.1
  • port (optional) is the TCP port number on which the Oracle Net listener is listening. If not specified, the default port number 1521 is assumed.
