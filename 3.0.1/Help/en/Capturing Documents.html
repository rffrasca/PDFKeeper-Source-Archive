Capturing Documents

PDF and XPS documents can be captured using the following methods:

  • From Windows Explorer, right click on one or more selected PDF/XPS document's and select Send to, and then PDFKeeper Document Capture.
  • From Web Browser/application, save PDF or XPS document to the Capture folder using the PDFKeeper Document Capture shortcut located in the My Documents folder on Windows XP or the Links folder on Windows Vista and later.
  • Configure Scanners/Document generator applications to save each PDF or XPS document to %USERPROFILE%\Local Settings\Application Data\PDFKeeper\Capture on Windows XP or %LOCALAPPDATA%\PDFKeeper\Capture on Windows Vista and later.

Note

  • The PDFKeeper Document Capture shortcuts are created when PDFKeeper is started and are removed when PDFKeeper is closed out.
  • When the Capture folder contains PDF document's, an icon matching the Document Capture tab icon will be visible on the right side of the PDFKeeper status bar. Clicking on this icon will select the Document Capture tab.
  • PDFKeeper Document Capture will only accept XPS or PDF documents, all other file types will be ignored and remain in the Capture folder until manually removed.
  • Each XPS document captured will be converted to PDF, and then deleted to the Windows Recycle Bin.
  • It is NOT recommended to configure the Windows Recycle Bin to delete files immediately as Document Capture will delete all preprocessed XPS and PDF files to the Recycle Bin.
