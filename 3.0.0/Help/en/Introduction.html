Introduction

The PDFKeeper client application is used to locate PDF document records in the database by Title, Author, Subject, Keywords, document notes, date and time added to the database, and any text within the PDF document.

The following tasks can be performed on any PDF document record that has been returned during a search:

  • View the document with bundled viewer;
  • Save the document to disk;
  • Edit and print document notes;
  • View the document keywords;
  • Delete selected database records.

In addition, you can edit the information properties for captured PDF documents and upload to the database.
