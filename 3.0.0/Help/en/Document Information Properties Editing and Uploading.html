Document Information Properties Editing and Uploading

Each PDF document listed in the Document Capture Queue list box can have its Information Properties edited, and then be uploaded to the database by performing the following:

  1. From the Document Capture Queue list box, select the PDF document. If the wrong document was selected or you need to cancel working with this document, select Deselect or press the Esc key. Note, each PDF document will remain in the Queue until it has been uploaded or has been removed using Windows Explorer.
  2. If necessary, you can view the PDF document by selecting View which will open it with the bundled, third-party PDF viewer.
  3. Enter the document Title.
  4. Enter or select the Author of the document. Note, the Author entry field is linked to the database to allow for selecting an existing Author.
  5. Enter or select the Subject of the document. Note, the Subject entry field is linked to the database to allow for selecting an existing Subject that is linked to the entered/selected Author.
  6. If necessary, enter document Keywords.
  7. Select Save. Note, this step will need to be repeated if changes are made following a Save. The original PDF document will be preserved in the Capture folder. A modified version of the PDF will be written in %USERPROFILE%\Local Settings\Application Data\PDFKeeper\CaptureTemp on Windows XP or %LOCALAPPDATA%\PDFKeeper\CaptureTemp on Windows Vista and later.
  8. If necessary, you can view the modified copy of the selected PDF document by selecting Preview which will open it with the bundled, third-party PDF viewer.
  9. If no further changes need to be made, you're ready to upload the modified copy of the selected PDF document to the database by selecting Upload. Note, after being uploaded, the modified copy of the selected PDF document will be deleted, and the original PDF document will be deleted to the Windows Recycle Bin and removed from the Document Capture Queue list box.

Note

  • USER password protected PDF documents are not supported by PDFKeeper. If you have the legal right to remove the password, you can find plenty of information on the internet to assist you.
  • OWNER password protected PDF documents are supported by PDFKeeper; however, when selected, the correct password will need to be entered when prompted. If you do not know the password and have the legal right to remove the password, you can find plenty of information on the internet to assist you.
  • If at anytime, you need to cancel working with the selected document, select Deselect or press the Esc key. Note, if a modified copy of the selected PDF does exist, it will be deleted. In addition, closing PDFKeeper will result in a loss of all modifications to the selected document.
  • Each PDF document will remain in the Queue until it has been uploaded or has been removed using Windows Explorer.
  • Switching between Document Capture and Document Search will not result in any loss of document modifications.
