User FAQ

General Information

What is PDFKeeper?
PDFKeeper is free, open source, PDF document storage software used for capturing, storing, and searching PDF documents using Oracle Database Express Edition.
Who should use PDFKeeper?
Anyone who wants to implement a low cost PDF document storage system and is capable of administrating a relational database.

Software Compatibility

Does PDFKeeper work on other Oracle databases?
Only Oracle Database Express Edition is supported. However, if you are a developer, experienced with both Oracle Database and VB.NET, you should be able to get it to function with some modification on any supported edition of the Oracle Database with Oracle Text enabled.
Will other databases be supported in the future?
It would be ideal to expand the scope of PDFKeeper to work with other editions of the Oracle Database and other databases that support PDF indexing. However, this will be dictated by need, time permitting.
Will PDFKeeper function on 64-bit versions of Windows?
PDFKeeper is a 32-bit application that should function on 64-bit Windows. However, it has only been tested on a 64-bit version of the Windows 8 operating system.

Troubleshooting

I get "ORA-03113: end-of-file on communication channel" following the upload of a PDF document. What can I do?
If your database schema was built using the Database Setup included with PDFKeeper 1.0.0 or 1.1.0, please refer to this Release Notes for the cause and workaround.

Other Questions

My question is not on this page. How do I find the answer?
First refer to the help system and guides included with PDFKeeper. If your question can't be answered there, search the project Discussions and Issues. If you still don't find your answer, you can start a new discussion thread or enter an issue.
