PDFKeeper 1.1.0: Release Notes

Product Release Information

Product: PDFKeeper
Release Number: 1.1.0
Release Date: 2010/04/29
Customer Support: For more information or support, please visit website or send an email to rffrasca@gmail.com

Introduction

This document contains the release notes for PDFKeeper version 1.1.0. The following sections describe the release in detail and provide late-breaking or other information that supplements the main documentation.

This is an upgrade release with some significant enhancements and fixes. Users of previous releases are encouraged to upgrade.

What's New?

  • Open Object Rexx 4.0.0 is now required.
  • Sumatra PDF has been updated, now at version 1.0.1
  • pdfinfo.exe from Xpdf has been updated, now at version 3.02pl4
  • Improvements to the product's quality and reliability. See "Recent Changes" below for details.
  • Additional software testing for this release was performed by nResult.

Installation and Upgrade Notes

Installation
See the installation instructions for full details.
The installer will remove prior versions of PDFKeeper, this includes the PDFKEEPER_UPLOAD environment variable. Following an upgrade, the Document Upload Folder will need to be selected.
Manifest
This release consists of the following items:
  • Release notes (this file)
  • Installation instructions / Quick start guide
  • Product installer binary
  • User guide
  • For product source code, please visit website
Minimum System Requirements
System Processor: Intel-compatible PC with a 32-bit processor
Free Disk Space: 5.4MB for the client-side components, 1.5MB for the server-side components; does not include data, temporary files, or prerequisites.
Operating System: Windows XP (32-bit)
Networking: TCP/IP; Client for Microsoft Networks and File and Printer Sharing for Microsoft Networks, if used in a Workgroup.
Existing Software:
Popular web browser to view documentation
Version Compatibility
The database schema is unchanged with this release.
