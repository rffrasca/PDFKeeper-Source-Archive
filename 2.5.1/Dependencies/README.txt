PDFKeeper requires the following third-party software to be in this
folder prior to building PDFKeeper:

	Sumatra PDF 1.9
	WWW: http://blog.kowalczyk.info/software/sumatrapdf/download-free-pdf-viewer.html
	ZIP: SumatraPDF-1.9.zip
	Required File: SumatraPDF.exe

	iTextSharp 5.2.0
	WWW: http://sourceforge.net/projects/itextsharp/files/itextsharp/iTextSharp-5.2.0/
	ZIP: itextsharp-all-5.2.0.zip
	Required File: itextsharp.dll
	Notes: Extract itextsharp.dll from 
	       itextsharp-all-5.2.0.zip\itextsharp-dll-core-5.2.0.zip

	Nini 1.1.0
	WWW: http://nini.sourceforge.net/download.php
	ZIP: Nini-1.1.0.zip
	Required File: Nini.dll
	Notes: Extract Nini.dll from Nini-1.1.0.zip\Nini\Bin\DotNet\2.0\Release

	CKill 1.0
	WWW: http://colinfinck.de/en/progs/ckill/
	ZIP: ckill_1.0.zip
	Required File: ckill.exe

Notes for files placed in this folder: 

	Make sure you "Unblock" each file by right clicking on each file in
	Explorer, select Properties, and then select "Unblock".
