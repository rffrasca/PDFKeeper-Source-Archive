PERFORM THE FOLLOWING STEPS PRIOR TO BUILDING PDFKEEPER:

1. Download SumatraPDF-1.9.zip from
   http://blog.kowalczyk.info/software/sumatrapdf/download-free-pdf-viewer.html
  
   - Extract SumatraPDF.exe from SumatraPDF-1.9.zip into this folder.
   - From Explorer, right click on SumatraPDF.exe and select Properties,
     and then select "Unblock".

2. Download itextsharp-all-5.1.3.zip from
   http://sourceforge.net/projects/itextsharp/files/
  
   - Open itextsharp-dll-core-5.1.3.zip located inside
     itextsharp-all-5.1.3.zip, and then extract itextsharp.dll into this
     folder.
   - From Explorer, right click on itextsharp.dll and select Properties,
     and then select "Unblock".
