Configuring Direct Upload sub-folders

Individual sub-folders can be configured (add, edit, and delete), each having its own preconfigured PDF Information Properties; these properties will be applied to each PDF file during the upload process.

To add a new sub-folder or edit an existing sub-folder, perform the following:

  1. If adding a new sub-folder select New; if editing an existing sub-folder, from the Folders list, select the folder, and then select Edit.
  2. Enter Folder Name.
  3. Select or specify the Title PreFill and if desired, check the box labeled Use existing Title, if not blank.
  4. Select or specify the Author PreFill and if desired, check the box labeled Use existing Author, if not blank.
  5. Select or specify the Subject PreFill and if desired, check the box labeled Use existing Subject, if not blank.
  6. If needed, specify the Keywords PreFill and if desired, check the box labeled Use existing Subject, if not blank.
  7. Select Save or press Enter to save the folder configuration; to discard the changes, select Discard or press Esc.

To delete an existing sub-folder, perform the following:

  1. From the Folders list, select the folder, and then select Delete.
  2. When prompted, select Yes or press Enter to delete the folder.

Note

  • Direct Upload will be suspended while the Direct Upload Configuration form is open.
