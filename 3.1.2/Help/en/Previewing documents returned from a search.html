Previewing documents returned from a search

To preview the first page of each document returned from a search without opening it using the bundled third-party viewer, select the document list view record on the Document Search tab, and then select the Document Preview tab. The first page of the PDF document will be displayed as an image.

The following functions can be performed using Document Preview:

  • Zoom in and out with the Zoom in and Zoom out buttons; or you can use the keyboard shortcuts Alt+I and Alt+O.
  • Preview the previous or next document in the result set using the Previous or Next buttons; or you can use the keyboard shortcuts Alt+P or Alt+N.
  • Open the previewed document using the bundled third-party viewer by double clicking on the preview image.
  • When switching back to the Document Search tab, the document that was being previewed will be the selected record in the document list view on the Document Search tab.

Notes

  • Each PDF document previewed and its preview image will remain cached on disk until PDFKeeper is closed out.
  • If your operating system supports the Encrypting File System (EFS), each PDF document previewed and its preview image will be encrypted. The following operating systems do not support EFS: Windows XP Home, Windows Vista Home Basic and Premium, and Windows 7 Starter, Home Basic and Premium.
