Known Problems and Workarounds

  • "REXX/SQL-1: Database Error - specified driver could not be loaded due to system error 1114 (Oracle in XEClient)"
    • WORKAROUND: The Oracle Database 10g Express Client ODBC driver requires MFC71.DLL, but Oracle did not include this file with the Oracle Database 10g Express Client. To correct, obtain a copy of MFC71.DLL from another application, and then copy it into a folder in the PATH. Note, this file is included with the Oracle Database 10g Express Edition Server.
  • The Title, Author, Subject, and Keywords summary fields in the PDF document cannot contain any "|" vertical bar characters. The Document Loader uses the "|" vertical bar as a separator character.
    • WORKAROUND: Remove any "|" vertical bar characters from the Title, Author, Subject, and Keywords summary fields with the PDFKeeper Summary Editor, prior to loading the PDF document into the database.
