Introduction

This document describes how to install and get started with PDFKeeper.

Minimal System Requirements

System requirements are described in the release notes.

PDFKeeper can be implemented for use on a single user system or in a workgroup. If implementing for use in a workgroup, it is recommended that the Oracle XE Database Server and the PDFKeeper server-side components be installed on a dedicated, Windows XP Professional system.

Installation

What other software must be installed first?
Before you can install this product, you must install the following packages:
  • Oracle Database 10g Express Edition (10.2.0.1) Server (Universal) on the system that will host the database.
  • Oracle Database 10g Express Client (10.2.0.1) on any system that will connect to the database remotely.
  • Open Object Rexx 3.2.0 on any system that will have the client-side components (Search and Summary Editor) installed.
How do I install PDFKeeper?
Please follow these steps:
  1. Download PDFKeeper-1.0.0.exe from here.
  2. As an Administrator, execute PDFKeeper-1.0.0.exe
    • At the welcome screen, click Next.
    • Review the license, select I accept the agreement, then click Next.
    • Select the destination folder or keep the default, then click Next.
    • Select the component(s) to install. For an explanation of the components, see the Getting Started section in the User FAQ.
    • Select the Start Menu folder or keep the default, then click Next.
    • Click Install.
    • Click Finish.
How can I uninstall PDFKeeper?
  1. Open Add or Remove Programs.
  2. Select PDFKeeper 1.0.0 from the list.
  3. Click Remove to start the uninstall.
    • Click Yes.
    • Click Ok.
  4. Close out of Add or Remove Programs.
What if I encounter problems?
Please see the troubleshooting section in the FAQ.
