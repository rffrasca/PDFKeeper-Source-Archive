PERFORM THE FOLLOWING STEPS PRIOR TO BUILDING PDFKEEPER:

- Download SumatraPDF-1.1.zip from
  http://blog.kowalczyk.info/software/sumatrapdf/download.html and
  extract SumatraPDF.exe from the archive into this folder.

- Download itextsharp-5.0.4-dll.zip from
  http://sourceforge.net/projects/itextsharp/files/ and
  extract itextsharp.dll from the archive into this folder.
