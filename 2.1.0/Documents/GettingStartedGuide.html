Introduction

This document describes how to get started with PDFKeeper.

PDFKeeper can be implemented for use on a single user system or in a workgroup. If implementing for use in a workgroup, it is recommended that the Oracle XE Database be installed on a dedicated system.

Getting Started

How can I get started using PDFKeeper?
  1. Please read the release notes for any late-breaking information.
  2. For new users'; read the Oracle Database 10g Express Edition documentation for instructions on how to install, configure, and administer an Oracle Express Edition Database.
  3. For new users'; on the system that will host the database, in addition to PDFKeeper, make sure the following software is installed:
    • Oracle Database 10g Express Edition (10.2.0.1) Server (Universal)
  4. On each system that the PDFKeeper application will be used on, make sure the following software is installed:
    • Microsoft .NET Framework 3.5 SP1
    • Oracle Data Access Components (ODAC) 11.2.0.1.2. Download the Xcopy version and follow the installation instructions in the zip file. Note, only the Oracle Data Provider for .NET 2.0 and it's dependency needs to be installed. To allow side-by-side installation with the Oracle Database Express Server or Client, make sure the oracle home specified during the ODAC installation is not XEClient or XE.
  5. Refer to the users' guide for configuring and using PDFKeeper. Note, if PDFKeeper was just upgraded from version 1.0.0 or 1.1.0, please refer to section 3. Database User Account Management for additional grants that need to be provided to existing database user accounts. No database configuration changes are required if upgrading from version 2.0.0.

Uninstalling PDFKeeper

How can I uninstall PDFKeeper?
  1. Make sure PDFKeeper is not in use.
  2. From each user account that used PDFKeeper, remove user settings by clicking Start | All Programs | PDFKeeper | Tools | User Settings Cleanup
  3. As an Administrator, open Add or Remove Programs
  4. Select PDFKeeper 2.1.0 from the list.
  5. Click Remove to start the uninstall.
    1. Click Yes.
    2. Click Ok.
  6. Close out of Add or Remove Programs.
