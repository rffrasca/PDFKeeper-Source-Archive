PERFORM THE FOLLOWING STEPS PRIOR TO BUILDING PDFKEEPER:

1. Download SumatraPDF-1.4.zip from
   http://blog.kowalczyk.info/software/sumatrapdf/download.html
   Extract SumatraPDF.exe from SumatraPDF-1.4.zip into this folder.

2. Download itextsharp-5.0.6-dll.zip from
   http://sourceforge.net/projects/itextsharp/files/
   Extract itextsharp.dll from itextsharp-5.0.6-dll.zip into this folder.
